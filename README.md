# Peppy Todo

### Instructions to run
- Install NodeJS and NPM
- Clone this repository
- `cd` to the project root folder and run `npm install`
- Run `npm run start`.

### Built with:
- React
- Redux
- TypeScript
- Pure CSS
- Styled Components
